# Hello DevOps — Simple Flask API

Простой проект на Python с использованием микрофреймворка Flask. Приложение реализует базовый REST API и предназначено для демонстрации работы веб-сервиса, проверки настройки окружения или использования в CI/CD пайплайнах.

## 📋 Описание

Приложение запускает веб-сервер, который отвечает на корневой URL (`/`) JSON-сообщением.

*   **API Endpoint:** `GET /`
*   **Ответ:** `{"message": "Hello, DevOps!"}`
*   **Порт:** `3600`
*   **Хост:** `0.0.0.0` (доступен извне контейнера/сети)

## 🛠 Стек технологий

*   **Python 3**
*   **Flask** — веб-фреймворк
*   **Pytest** — для запуска модульных тестов

## 🚀 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone <URL_вашего_репозитория>
cd <папка_проекта>
```

2. Установка зависимостей
Рекомендуется использовать виртуальное окружение.
Linux/macOS:
```bash
python3 -m venv venv
source venv/bin/activate
pip install flask pytest
```

Windows:
```bash
python -m venv venv
.\venv\Scripts\activate
pip install flask pytest
```

3. Запуск приложения
```bash
python app.py
```

После запуска сервер будет доступен по адресу: http://localhost:3600
🧪 Тестирование
Проект включает в себя автоматические тесты для проверки доступности API и корректности возвращаемых данных.
Для запуска тестов выполните команду:
```bash
pytest
```

Ожидаемый результат:
```bash
test_app.py .                                                            [100%]
1 passed in 0.05s
```

📂 Структура файлов
app.py — Основной код приложения (Flask).
test_app.py — Модульные тесты (Pytest).
